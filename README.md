# Toyota US Sustainability Website — Wireframe Prototype

Early-stage wireframe prototype for the Toyota North America sustainability landing page project.

## Pages

- `index.html` — Main sustainability landing page
- `newsroom.html` — Newsroom / stories listing
- `story.html` — Individual story template (Nature Positive example)
- `reports.html` — Reports and data downloads

## Shared assets

- `shared.css` — Global stylesheet used across all pages

## Deployment

This project deploys automatically to Vercel. The `vercel.json` config enables clean URLs and disables trailing slashes.

| Page | URL |
|------|-----|
| Landing page | `/` |
| Newsroom | `/newsroom` |
| Story template | `/story` |
| Reports | `/reports` |

## Status

Wireframe prototype — March 2026. Content is illustrative. Real images, confirmed URLs, and final copy to be integrated in subsequent phases.
